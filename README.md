⚠️ IMPORTANT

- SESSION_ID
- OWNER
- PREFIX
- MODE

before it will work correctly.

---

## Features

- Multi-session support
- Session loading from Pastebin
- Plugin-based command system
- Clean, step-by-step startup logs
- Local PM2 support
- Render compatible

---

## Requirements

- Node.js 20+
- A valid Baileys session stored on Pastebin
- Basic knowledge of environment variables

---

## Setup

1. Fork this repository
2. Create a `.env` file
3. Add your `SESSION_ID`
4. Deploy on Render or run locally

---

## Notes

- This project intentionally avoids heavy features
- No database is included
- No QR or pairing flow
- No menu generator by default

This codebase is meant to be extended carefully.
